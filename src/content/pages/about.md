# About This Site

This is a static site built with modern web technologies.

## The Stack

- **Frontend**: Astro with TypeScript
- **CMS**: Decap CMS (formerly Netlify CMS)
- **Hosting**: Cloudflare Pages
- **CDN**: Cloudflare's global network

## Why This Stack?

1. **Performance** - Static sites are fast by default
2. **Security** - No server-side vulnerabilities
3. **Cost** - Free hosting on Cloudflare Pages
4. **Developer Experience** - Great tooling and documentation
